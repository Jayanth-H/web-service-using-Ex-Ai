ABOUT OUR PROJECT

The "Web Service Classification and Recommendation" project aims to enhance the process of selecting and recommending web services using machine learning techniques. The project is divided into two primary components:

Classification Model

This model classifies web services into quality categories—Bronze, Silver, Gold, and Platinum—based on key Quality of Service (QoS) attributes like:
  • Response time
  • Availability
  • Reliability
  • Throughput
The model is built using several machine learning algorithms, including Decision Trees, Support Vector Machines (SVM), Logistic Regression, K-Nearest Neighbors (KNN), Naive Bayes, Random Forest, Multi-Layer Perceptrons (MLP), and XGBoost. Explainable AI (LIME) is also integrated to make classification results transparent and easy to understand.

Recommendation Model

Utilizing the K-Nearest Neighbors (KNN) algorithm, this model recommends the top 10 most relevant web services based on user-inputted QoS attributes. It measures similarity between input and existing services to generate tailored, accurate recommendations.

User Interface

The platform provides a user-friendly interface where users can:
  • Upload web service data
  • View classification results
  • Receive personalized service recommendations
